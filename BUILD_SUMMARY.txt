PROJECT OVERVIEW:
================

A voice-enabled agentic AI assistant for dental clinics, delivered as a web
chat widget. The assistant uses Hermes LLM (via OpenRouter) with function-
calling to intelligently handle:

1. APPOINTMENTS: Book, reschedule, cancel dental appointments
2. FAQ: Answer dental questions with voice responses

Built with: FastAPI + SQLAlchemy + Whisper + gTTS + JavaScript

HOW IT WORKS (User Journey):

1. User opens: http://your-app-url
2. Selects flow: "Book Appointment" or "Ask Questions"
3. Option A - Type:
   - Enters text: "Book a cleaning for July 25 at 2 PM"
   - Sends message
4. Option B - Speak:
   - Clicks mic button
   - Speaks naturally
   - Audio transcribed via Whisper
5. Backend processes:
   - Message sent to Hermes LLM
   - Hermes reads tool definitions, decides which to call
   - System executes: book_appointment("User", "2026-07-25", "14:00", "cleaning")
   - Database updated with new appointment
   - Hermes generates human-friendly response
6. Frontend displays:
   - Confirmation text: "I've booked your appointment..."
   - (If FAQ) Audio plays with answer read aloud

API ENDPOINTS:

1. GET /
   Returns: Chat widget HTML
   
2. GET /api/config
   Returns: {"app_name": "...", "features": {...}}
   
3. POST /api/chat
   Body: {"text": "user message", "context": [...]}
   Returns: {"assistant_reply": "...", "tool_used": "...", "audio_url": "..."}
   
4. POST /api/transcribe
   Body: multipart/form-data with audio file
   Returns: {"success": true, "text": "transcribed text"}
   
5. POST /api/synthesize
   Body: {"text": "text to speak"}
   Returns: {"success": true, "audio_url": "/static/reply_xxx.mp3"}
   
6. GET /health
   Returns: {"status": "ok", "service": "Dental Web Agent"}

DEPLOYMENT TO RENDER (5 minutes):

1. Commit to Git:

   OPENROUTER_API_KEY=sk-or-v1-your-key
   OPENROUTER_MODEL=meta-llama/llama-2-7b-chat
   DATABASE_URL=sqlite:///./dental.db
   STT_MODEL=base
   TTS_LANGUAGE=en

4. Deploy:
   Click "Create Web Service"
   Wait 3-5 minutes for build
   Get public URL: https://dental-web-agent-xxx.onrender.com

5. Test:

   Open https://your-url in browser
   Test appointments and FAQ flows

TROUBLESHOOTING:

Q: Whisper/FFmpeg errors?
A: Install FFmpeg:
   - Windows: choco install ffmpeg
   - Mac: brew install ffmpeg
   - Linux: sudo apt-get install ffmpeg

Q: OpenRouter API errors?
A: Verify:
   - Key in .env: OPENROUTER_API_KEY=sk-or-v1-xxx
   - Not empty
   - Valid at https://openrouter.ai/

Q: Microphone not working?
A: Microphone requires HTTPS (Render URLs are HTTPS ✓)
   Works on https://your-url
   Doesn't work on http://localhost:8000 (local testing workaround: use text)

Q: Slow first request?
A: Render free tier spins down after 15 min inactivity
   First request = 20-30s (app startup + Whisper model loading)
   Subsequent = 2-5s (normal)
